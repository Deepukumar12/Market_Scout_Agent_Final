01

The Entry Point app/api/scan.py

When you click "Start Scan" in the React UI, a Pydantic-validated request is sent to the FastAPI backend. This file acts as the gateway, checking your authentication (JWT) and passing the company name to the orchestrator.

  • FastAPI
  • OAuth2
  • JWT Authentication
02

Query Planning app/services/gemini_client.py

The system doesn't just search the company name. It uses Gemini 1.5 Pro to "think" about what technical data is needed. It generates 3-5 specific, high-intent search queries based on the target company and time window.

Example: "Google" ➔ "Google Cloud API release notes February 2026"

LLM Interaction 50-200 Tokens
03

Web Search & Scrape search_service.py & scraper_service.py

The Serper API executes the planned queries. Then, ZenRows (a headless browser engine) navigates to each URL. It bypasses bot protections, handles JavaScript rendering, and pulls the raw page content.

  • Serper API (Google Index)
  • ZenRows (Proxy/Headless)
  • BeautifulSoup4
04

Refining The Signal content_cleaner.py

Raw HTML is messy. The Content Cleaner strips away ads, headers, footers, and tracking scripts. Then, the Time Filter checks the page metadata. If the article is not technical or is older than the user-specified window, it is instantly deleted to save processing cost.

05

Intelligence Synthesis app/models/scan.py

The final mass of cleaned text is sent to the LLM. It extracts specific, technical updates and formats them into a strict JSON Schema. This ensures the frontend doesn't break and can render high-quality charts and bullet points automatically.

Heavy Extraction 5,000 - 30,000 Tokens
